HTTP 和 HTTPS 的区别如下：

1. HTTPS 是在 HTTP 的基础上加入了 SSL/TLS 加密协议，能够对传输的数据进行加密，防止数据在传输过程中被窃取或篡改。

2. HTTP 使用 80 端口，HTTPS 使用 443 端口。

3. HTTPS 需要向证书颁发机构（CA）申请 SSL 证书，浏览器会验证证书的有效性。

4. HTTPS 的安全性更高，现代浏览器对 HTTP 网站会显示"不安全"警告。

5. HTTPS 在建立连接时需要进行 SSL 握手，会比 HTTP 多一些延迟，但对用户体验影响较小。

6. SEO 方面，搜索引擎会优先收录 HTTPS 网站。
